## Academic Project Overview

This project was initially developed as an academic project during my postgraduate studies to understand the fundamentals of Data Analysis and Machine Learning. The goal of the project was to analyze air quality data and predict Air Quality Index (AQI) values using different Machine Learning algorithms.

As a beginner-level Machine Learning project, it helped me gain practical experience in data preprocessing, exploratory data analysis, data visualization, model training, model evaluation, and deployment. The project was later improved and deployed using Streamlit to provide a simple web-based interface for AQI prediction.

## Objectives

* Understand the basics of Machine Learning workflows.
* Analyze air quality and pollutant data.
* Calculate and classify Air Quality Index (AQI).
* Train and compare multiple regression models.
* Deploy the trained model using Streamlit.
